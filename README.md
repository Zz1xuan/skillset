# 🧠 My Skills Backup

一个用于存放个人常用技能（skills）的仓库，仅用于日常工作与生活中的知识备份与快速查阅。

---

## 📦 已收录技能

- `tool/har-ad-analysis`：HAR 广告分析与规则生成 skill
- `tool/har-task-automation-script`：HAR 任务自动化脚本生成 skill
- `tool/a-share-stock-picker`：A 股选股 skill，来源标注为 Linux.do 分享包

---

## 📌 项目说明

本仓库主要用于整理和记录我在工作与生活中使用到的各种技能与经验，包括但不限于：

- 🖥️ 开发相关（如 Git、脚本、工具配置等）
- ⚙️ 效率工具使用（如终端、编辑器、自动化）
- 🧩 常见问题解决方案（踩坑记录）
- 📚 零散但实用的知识点

> 内容以“能用、好查、可复用”为优先，不追求系统性教程。

---

## 🚀 使用方式

- 按目录查找对应技能
- 使用关键词搜索（推荐直接用 GitHub 搜索）
- 持续补充与更新内容

---

## 📝 更新原则

- 优先记录**自己实际用到的内容**
- 保持内容**简洁清晰**
- 避免无意义复制粘贴
- 以“未来的自己能快速看懂”为目标

---

## ⚠️ 注意

- 本仓库内容主要服务于个人使用
- 不保证内容完整性或通用性
- 如有错误或过时内容，会随使用逐步修正

---

## 📎 备注

这是一个长期积累的“个人知识仓库”，更像是一个随手记录的工具箱，而不是系统教程或最佳实践集合。
